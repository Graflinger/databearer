## Technische oder notwendige Daten

Durch das Nutzen der Website werden folgende notwendige Daten erfasst:

- Verwendete IP-Adresse
- Besuchte Webseite
- Verwendeter Browser
- Verwendetes Betriebssystem
- Uhrzeit zum Zeitpunkt des Zugriffes
- Menge der gesendeten Daten in Byte
- Quelle/Verweis, von welchem Sie auf die Seite gelangten

## Speichern von personenbezogenen Daten

Für den Anmeldeprozess werden die Daten (Email, Name) gespeichert, sobald die Daten vom User freiwillig eingegeben werden, um einen Account zu generieren.

Falls eine kostenpflichtige Mitgliedschaft aktiviert wird, werden die Daten an den Zahlungsdienstleister Stripe weitergeleitet, die für die Zuordnung des Accounts notwendig sind.

Die Email-Adresse, die bei dem Account hinterlegt ist, wird für Newsletter benutzt werden. Die Newsletter können auch abbestellt werden.

Der Account kann jederzeit auf Anfrage gelöscht werden und es werden keine Daten gespeichert nach der Löschung.

## Webseiten-Analyse mit Matomo

Matomo wird als Tool benutzt, um Statistiken zur Websitenutzung zu generieren. Hierbei werden keine Cookies verwendet, keine personenbezogenen Daten gespeichert (IP Adresse) und alle Daten werden in der EU verarbeitet. Zudem wird auf im Browser voreingestellte Trackingverbote Rücksicht genommen.

Weitere Informationen: [GDPR - Matomo Analytics](https://matomo.org/gdpr-analytics/)
