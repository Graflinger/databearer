# Web Performance Optimization Tips

Website speed directly impacts user experience and SEO rankings.

## Quick Wins

- Optimize images
- Minimize CSS and JavaScript
- Enable caching
- Use a CDN

Small optimizations can lead to significant improvements in load times and user satisfaction.
