# Modern CSS Techniques

CSS has evolved dramatically, offering powerful new features for web developers.

## New CSS Features

- CSS Variables (Custom Properties)
- Flexbox and Grid
- Container Queries
- CSS Subgrid

These modern techniques allow for more maintainable and flexible stylesheets.
